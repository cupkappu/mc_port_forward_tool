# Fabric 1.20.1 / 1.21.1 E2E Compatibility Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development for code changes and superpowers:verification-before-completion before claiming a target version works. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build MC Transport Dialer for pure Fabric Minecraft `1.20.1` and `1.21.1`, then verify TCP byte transport with real Fabric server/client E2E tests.

**Architecture:** Keep transport core independent from Minecraft/Fabric APIs. Select one version-specific Fabric adapter source root per Gradle invocation with `-PtargetMinecraft=1.20.1` or `-PtargetMinecraft=1.21.1`.

**Tech Stack:** Java 17 for Minecraft `1.20.1`, Java 21 for Minecraft `1.21.1`, Fabric Loader `0.19.3`, Fabric API `0.92.9+1.20.1` / `0.116.12+1.21.1`, Fabric Loom, JUnit 5.
